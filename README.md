# Wakey

Wakey is a lightweight macOS menu bar app for keeping your Mac awake without leaving sleep prevention on all day. It is built for short focused sessions, recurring active hours, and app-based triggers.

## What It Does

- Start and stop wake control from the menu bar.
- Use quick timers for `1 minute`, `5 minutes`, `10 minutes`, or `Forever`.
- Keep your Mac awake during scheduled hours.
- Automatically stay awake while selected apps are running.

## Install

Download the latest version from the [GitHub Releases](https://github.com/kzkit/wakey/releases/latest) page.

1. Open the downloaded release file.
2. Move `Wakey.app` to your `Applications` folder if needed.
3. Launch Wakey from `Applications`.

## Usage

1. Open Wakey from the menu bar.
2. Choose a quick timer or keep-awake mode.
3. Use Settings to configure scheduled hours or app-aware triggers.
